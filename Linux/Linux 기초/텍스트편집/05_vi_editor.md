# vi / vim 편집기

## 모드

- **명령 모드**: 기본 모드, 이동·복사·삭제 등
- **입력 모드**: 텍스트 입력 (`i`, `a` 등)
- **마지막 줄 모드**: 저장·종료 등 (`:`)

## 모드 전환

| 키 | 동작 |
|----|------|
| i | 현재 위치에서 입력 모드 |
| a | 커서 다음에서 입력 모드 |
| o | 다음 줄에 새 줄 추가 |
| Esc | 명령 모드로 복귀 |
| : | 마지막 줄 모드 |

## 기본 이동

| 키 | 동작 |
|----|------|
| h, j, k, l | 왼쪽, 아래, 위, 오른쪽 |
| w | 다음 단어 처음으로 |
| b | 이전 단어 처음으로 |
| 0 | 줄 맨 앞 |
| $ | 줄 맨 뒤 |
| gg | 파일 맨 처음 |
| G | 파일 맨 끝 |
| 5G | 5번째 줄로 |

## 삭제·복사·붙여넣기

| 키 | 동작 |
|----|------|
| x | 한 글자 삭제 |
| dd | 한 줄 삭제 |
| 3dd | 3줄 삭제 |
| yy | 한 줄 복사 |
| p | 아래에 붙여넣기 |
| P | 위에 붙여넣기 |

## 검색

| 키 | 동작 |
|----|------|
| /검색어 | 아래로 검색 |
| ?검색어 | 위로 검색 |
| n | 다음 결과 |
| N | 이전 결과 |

## 마지막 줄 모드 (:)

| 명령 | 동작 |
|------|------|
| :w | 저장 |
| :q | 종료 |
| :wq | 저장 후 종료 |
| :q! | 저장 안 하고 강제 종료 |
| :set number | 줄 번호 표시 |
| :10 | 10번째 줄로 이동 |

## 자주 쓰는 조합

```
i          → 입력 → Esc → :wq     # 수정 후 저장 종료
dd         # 줄 삭제
yy + p     # 줄 복사
/u + n     # 'u' 검색 후 다음
```
