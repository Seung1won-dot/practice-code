# Linux

## Linux란?

Linux는 1991년 리누스 토르발스(Linus Torvalds)가 개발한 오픈소스 운영체제 커널입니다. 서버, 임베디드, 클라우드, 슈퍼컴퓨터 등 전 세계 시스템의 대부분을 구동합니다.

## Linux의 특징

### 1. 오픈소스
- 소스 코드가 공개되어 있어 누구나 수정·배포 가능
- 무료로 사용 가능
- 커뮤니티 기반 개발

### 2. 멀티유저·멀티태스킹
- 여러 사용자가 동시에 사용
- 여러 프로세스가 동시에 실행
- 사용자별 권한 관리

### 3. 안정성과 보안
- 서버 환경에 최적화
- 정기적인 보안 패치
- 방화벽, SELinux 등 보안 기능

### 4. 명령줄 중심
- CLI(Command Line Interface)로 대부분의 작업 수행
- 스크립트를 통한 자동화 용이
- GUI 없이도 모든 작업 가능

### 5. 다양한 배포판
- Ubuntu, CentOS, Debian, Fedora, Arch 등
- 용도에 맞는 배포판 선택 가능
