## HTML이란

HTML은 하이퍼(Hyper) 텍스트(text) 마크업(markup) 언어(language)의 준말로,
웹 페이지의 구조를 정의하는 웹을 위한 마크업 언어이다.

### 웹(Web)이란

웹이란 인터넷을 통해 연결된 사용자들이 서로의 정보를 공유할 수 있는 공간을 의미한다.
정확히 말하자면, 월드 와이드 웹(World Wide Web)을 의미하며, www나 W3라고도 부른다.

인터넷과 같은 의미로 사용되고 있지만, 정확히는 인터넷상에서 가장 인기 있는 하나의 서비스일 뿐이다.
스카치 테이프가 테이프의 대명사가 된 것처럼,
웹과 인터넷, 두 단어가 혼용되고 동일시될 정도로 웹은 인터넷에서 가장 큰 부분을 차지하고 있다.

### 웹의 특징

위에서 HTML은 하이퍼텍스트 마크업 언어의 준말이라고 했다.
그렇다면 하이퍼텍스트는 뭐고 마크업 언어는 뭘까?

웹은 인터넷 상에서 텍스트나 그림, 소리, 영상 등 여러 멀티 미디어 정보를 하이퍼텍스트 방식으로 연결, 제공한다.

여기서 하이퍼텍스트(HyperText)란 문서 내부에 또 다른 문서로 연결되는 ‘참조’를 집어 넣는 것으로 웹 상에 존재하는 여러 문서끼리 서로를 ‘참조’할 수 있는 기술을 의미한다.
이때 문서 내부에서 또 다른 문서로 연결되는 참조를 하이퍼링크(hyperlink)라고 부른다.

### 웹의 구성

HTML이라는 언어를 사용하여 작성된 하이퍼텍스트 문서를 웹 페이지(Web-page)라고 부른다.
이런 웹 페이지들 중 서로 관련 있는 웹 페이지들의 집합을 웹 사이트(Web-site)라고 부른다.

웹은 이렇게 작성된 여러 문서, 즉 웹 페이지들 간의 하이퍼링크를 통해 서로 연결, 구성된다.

웹 서핑(web-surfing: 사용자가 웹페이지의 하이퍼링크를 통해 다른 웹페이지로 이동하는 것을 웹 서핑이라 부른다.

웹 브라우저(web-browser) : 사용자가 웹페이지를 검색하기 위해 사용하는 프로그램을 웹 브라우저라고 부른다.

### 마크업 언어

마크업 언어(Markup Language)는 인터넷에 검색하면 아래와 같이 정의되어 있다.

> **마크업 언어**(markup 言語, markup language)는 [태그](https://ko.wikipedia.org/wiki/%ED%83%9C%EA%B7%B8_(%EC%A0%95%EB%B3%B4)) 등을 이용하여 문서나 데이터의 논리구조를 명시하기 위한 규칙들을 정의한 언어의 일종
> 

여기서 눈여겨봐야 할 부분은 “문서나 데이터의 논리구조를 명시하기 위한 규칙들을 정의한 언어”라는 부분이다.
어떠한 데이터를 화면에 구조적으로 표시하기 위해 사용되는 언어라는 것이다.
하지만 일반적으로 데이터를 보여주기 위한 정도로만 사용되기에 프로그래밍 언어와는 구별된다.

### 마크다운 언어

마크업 언어의 일종으로, 간단한 텍스트들과 특정 기호의 조합을 통해 쉽게 글자크기를 바꾸거나 글자를 두껍게, 기울이기, 밑줄, 취소선 등을 사용할 수 있는 언어이다.

주로 위키나 깃허브와 같은 곳에서 문서를 설명하기 위한 언어로 사용된다.
마크업 언어를 기반으로 했기 때문에 역시나 프로그래밍적인 부분은 없다.
