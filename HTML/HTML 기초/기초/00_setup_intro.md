# HTML 개발 환경 설정

## HTML 개발 도구

### 텍스트 에디터
- **VS Code**: 가장 인기 있는 에디터, HTML/CSS/JS 확장 지원
- **Sublime Text**: 가벼운 에디터
- **Atom**: GitHub 개발 에디터
- **Notepad++**: Windows용 경량 에디터

### 통합 개발 환경 (IDE)
- **WebStorm**: JetBrains의 웹 개발 IDE
- **Visual Studio**: Microsoft IDE
- **Brackets**: Adobe 개발 에디터 (실시간 미리보기)

### 브라우저 개발자 도구
- **Chrome DevTools**: F12 또는 Ctrl+Shift+I
- **Firefox Developer Tools**
- **Safari Web Inspector**
- **Edge DevTools**

## 기본 HTML 파일 생성

1. `.html` 확장자로 파일 저장
2. 브라우저에서 직접 열거나 Live Server 확장으로 실행
3. VS Code의 "Open with Live Server"로 실시간 미리보기 가능

## .html 파일 실행 방법

- **로컬**: `파일명.html` 더블클릭 또는 브라우저에서 파일 열기
- **Live Server**: VS Code 확장 설치 후 우클릭 → Open with Live Server
- **터미널**: `npx serve .` 또는 `python -m http.server` 로 로컬 서버 실행

## HTML 버전

- **HTML5**: 현재 표준 (2014년)
- DOCTYPE 선언: `<!DOCTYPE html>`로 HTML5 사용 선언
