template과 slot

template은 렌더링되지 않는 콘텐츠 템플릿입니다. JavaScript로 활성화할 때까지 비활성 상태로 유지됩니다.

slot (Web Components)

Custom Element와 Shadow DOM에서 slot은 자식 콘텐츠를 삽입하는 플레이스홀더입니다.

// 사용 예 (Custom Element 정의 시)
class MyCard extends HTMLElement {
    connectedCallback() {
        this.attachShadow({mode: 'open'}).innerHTML = `
            <div class="card">
                <h3><slot name="title"></slot></h3>
                <p><slot>기본 내용</slot></p>
            </div>
        `;
    }
}
// <my-card><span slot="title">제목</span>본문</my-card>
