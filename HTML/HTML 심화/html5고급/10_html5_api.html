HTML5 API 예제

1. 비디오 (video)

source로 여러 포맷을 지원하며, 브라우저가 지원하는 첫 번째 포맷을 사용합니다.

2. 오디오 (audio)

3. 캔버스 (canvas)

2D 그래픽을 JavaScript로 그립니다. 차트, 게임, 이미지 편집 등에 사용됩니다.

4. SVG (Scalable Vector Graphics)

벡터 그래픽. 확대해도 깨지지 않습니다.

5. Geolocation, Web Storage 등

HTML5는 Geolocation(위치), LocalStorage, SessionStorage, Web Workers, WebSocket 등 다양한 API를 제공합니다. JavaScript와 함께 사용합니다.
