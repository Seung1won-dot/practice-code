# 분할 정복 알고리즘

## 개념

분할 정복(Divide and Conquer)은 문제를 작은 부분 문제로 나누어 해결한 후, 그 결과를 결합하여 원래 문제를 해결하는 알고리즘 설계 기법입니다.

## 단계

1. **분할(Divide)**: 문제를 더 작은 하위 문제로 나눕니다.
2. **정복(Conquer)**: 하위 문제를 재귀적으로 해결합니다.
3. **결합(Combine)**: 하위 문제의 해를 결합하여 원래 문제의 해를 구합니다.

## 특징

- 재귀적 구조를 가집니다.
- 병렬 처리에 적합합니다.
- 시간 복잡도가 효율적일 수 있습니다.

## 대표적인 예시

- 퀵 정렬
- 병합 정렬
- 이진 탐색
- 하노이 탑
